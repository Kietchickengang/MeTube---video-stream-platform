# MeTube Frontend Help Content

This README is a frontend-accessible copy of the project documentation shown in the Help page.

## Features

- Search videos by keyword.
- Watch history with per-user persistence.
- Save videos to Watch Later for each user.
- Like videos per account and view liked videos history.
- Subscribe to channels and see your subscribed channels.
- Upload videos and manage your own uploads.
- Toggle dark/light theme across the entire interface.
- View debug and error reports from the system.

## Usage

1. Sign in or register to activate personalized features.
2. Use the search box in the top navigation to find videos.
3. Open a video to like it, subscribe to the channel, or save it for later.
4. Visit `Watch Later`, `Liked Videos`, and `Subscriptions` pages to manage your saved content.
5. Open `Settings` to switch between Dark and Light themes.
6. Visit `Help` to read the documentation and `Report History` to inspect system logs.
